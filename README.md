# NeuralNetArthroplasty
Code, logic, and rationale for predicting total hip arthroplasty and total knee arthroplasty outcomes (length of stay and cost) using clinical databases

Dependencies:
Python 3.5.1  
TensorFlow 1.11.0  
Keras 2.2.4  
Sci-kit Learn 0.20.0  
Numpy 1.10.4  
Scipy 1.1.0  
Pandas 0.22.0  
Matplotlib 2.1.2
